# Web del Proyecto Aegis

Este repositorio contiene el código fuente del sitio web promocional y de soporte para Aegis, el plugin de gestión de permisos para Minecraft.

Esta web ha sido desarrollada como parte del Proyecto Intermodular de 2º DAM

## Estructura 

El sitio web consta de tres secciones principales:

* **Inicio (`index.html`):** Página de presentación donde se muestran las características principales del plugin (Interfaz GUI, Herencia Visual, etc.), requisitos técnicos y enlaces al repositorio de descarga.
* **Gestión de Incidencias (`incidencias.html`):** Formulario diseñado para que los usuarios reporten bugs encontrados en el servidor. Permite detallar los pasos para reproducir el error y adjuntar logs.
* **Feedback (`feedback.html`):** Encuesta de satisfacción para valorar la usabilidad de la interfaz gráfica y recoger sugerencias de mejora (NPS).
