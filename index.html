Aegis

Sistema de gestión de permisos visual e interactivo para Minecraft

Estado del Proyecto: Actualmente en fase de desarrollo (Alpha). Compatible con Spigot/Paper 1.21+.

Características Principales

Interfaz GUI

Crea, edita y borra grupos de permisos usando inventarios. Ideal para administradores que prefieren lo visual.

Herencia Visual

Establece jerarquías de rangos (Moderador -> Admin) arrastrando ítems dentro del menú.

Buscador Inteligente

Integración con la API de Bukkit para sugerirte los nodos de permisos de tus otros plugins.

Datos Híbridos

Tú eliges: Configuración simple en YAML o bases de datos MySQL/SQLite para redes grandes.

Requisitos Técnicos

  • Java: Versión 21 (LTS).
  • Servidor: Spigot, Paper o purpur (1.21 o superior).
  • Base de Datos (Opcional): MySQL si necesitas sincronización entre servidores.

Descarga y Prueba

El proyecto es de código abierto. Puedes descargar la última *build* desde nuestro repositorio o contribuir al código.
